Personal Finance Assistant

The  Personal Finance Assistant is a Python project that helps users manage their personal finances by tracking income, expenses, and budgets, while also providing basic predictions on future spending.

Features
Add and track income
Add and categorize expenses
View income and expense summaries
Set and monitor monthly budgets
Get alerts when exceeding budget
Predict when budget will be exhausted
View monthly financial report

Technologies Used
Python
Streamlit
matplotlib and pandas

How It Works
User inputs income and expenses
Data is stored in an SQLite database
The system calculates:
Total income
Total expenses
Remaining budget
